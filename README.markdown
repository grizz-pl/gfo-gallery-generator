gfo gallery generator by Witold Firlej
=============================================
ver. 0.25

Requirements
------------

*  python 2.6
*  [highslide](http://highslide.com/)

HowTo
-----
1.  make static/ directory
2.  put your logo.png and intro.png file into it - standard width 750px
3.  modify gfo.conf file
4.  run ./gfo.py
